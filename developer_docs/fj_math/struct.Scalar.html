§A rational, finite scalar value

Scalar is a wrapper around f64 which guarantees that the contained value is both rational and finite. This allows Scalar to provide implementations of Eq, Ord, and Hash; enabling Scalar (and types built on top of it), to be used as keys in various sets and maps.
