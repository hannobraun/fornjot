1//! # Operations to build objects
2//!
3//! All object structs have constructors, but building even simple shapes by
4//! just using those is pretty difficult. This is why the traits in this module
5//! exist. They build on top of those low-level constructors, providing a more
6//! rich set of operations to build geometry.
7//!
8//!
9//! ## Wrapper Structs
10//!
11//! Many of the the trait methods return the object that is being built
12//! directly, but others return a wrapper struct. An example of this is
13//! [`BuildFace::triangle`], which returns [`Polygon`] instead of returning the
14//! face directly.
15//!
16//! These wrapper structs are designed to provide convenient access not only to
17//! the top-level object itself, but also to the other objects that make up its
18//! components.
19

23mod half_edge;
