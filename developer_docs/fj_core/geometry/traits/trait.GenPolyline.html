Expand description

§Generate polylines, the uniform representation of curve geometry

This trait provides a generic and uniform interface to curve geometry. It is implemented by types that represent specific kinds of curve geometry.

It is generic over the dimensionality of the generated polyline. Typically, two variants should be implemented per curve geometry type:

  • GenPolyline<2> for surface-local geometry.
  • GenPolyline<3> for global 3D geometry.
