§Sweep the Cycle

Sweep the cycle into a set of connected faces. Each half-edge in the cycle is swept into a face, meaning all resulting faces form a connected set of side walls.

Requires the surface that the half-edges of the cycle are defined in, and optionally the color of the created faces.

There is no face at the “top” (the end of the sweep path), as we don’t have enough information here to create that. We just have a single cycle, and don’t know whether that is supposed to be the only cycle within a potential top face, or whether it’s an exterior or interior one.

For the same reason, there also is no “bottom” face. Additionally, whether a bottom face is even desirable depends on the context this operation is called in, and therefore falls outside of its scope.
