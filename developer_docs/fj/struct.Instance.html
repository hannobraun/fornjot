pub fn process_model<M>(&mut self, model: &M) -> Result
where for<'r> (&'r M, Tolerance): Triangulate, for<'r> &'r M: BoundingVolume<3>,

Export or display a model, according to CLI arguments

This function is intended to be called by applications that define a model and want to provide a standardized CLI interface for dealing with that model.

This function is used by Fornjot’s own testing infrastructure, but is useful beyond that, when using Fornjot directly to define a model.

where for<'r> (&'r M, Tolerance): Triangulate, for<'r> &'r M: BoundingVolume<3>,

Process a model with pre-parsed arguments

This function is similar to Self::process_model, but accepts pre-parsed arguments instead of parsing them from the command line. This is useful when you want to extend the standard arguments with your own parameters.
